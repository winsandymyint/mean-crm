Getting started

Before we start, let's take a look on the folder structure of the theme. All theme files are placed in the frontend folder. This folder consists of 8 more folders for different types of theme assets: 

admin/
├── bootstrap/ //Bootstrap Core
├── css/ 
├── fonts/
├── images/
├── js/
├── less/
├── revolution/ // All assets for revolution slider
└── html/ //All html files are located inside this folder

These are 2 main files that you have to add to your page.

<!-- Core css -->
<link href="css/app.css" rel="stylesheet">


<!-- Core js -->
<script src="js/app.js"></script>

CSS

This theme is built on Twitter's Bootstrap Framework. This documentation included some of the information about Bootstrap Framework, but If you are not familiar with Bootstrap, it's highly recommended that you learn directly from Twitter's Bootstrap documentation.

It's also built with Less css, all less files are located under less/ directory.

Theme color

The default primary color in this theme is #ffb038 (Yellow). You can change this color to your favorite color by updating value of less variable @main-color). This variable is located under less/variable.less

Learn more about less css
